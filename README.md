## 🚀 Fonctionnalités Clés
- **Monitoring en Temps Réel :** Ticker de prix dynamique et graphiques interactifs (via Recharts).
- **Intelligence Artificielle :** 
  - 🤖 **Résumé IA :** Synthèse automatique des articles en 3 points clés.
  - 💬 **Chatbot Financier :** Assistant conversationnel pour répondre aux questions sur les marchés.
- **Données Boursières :** Statut d'ouverture des bourses mondiales avec compte à rebours en temps réel.
- **Expérience Utilisateur Premium :** Mode sombre (Style TradingView), système de favoris et prise de notes personnelles.

## 🛠️ Stack Technique
- **Framework :** [Next.js 16](https://nextjs.org/) (App Router)
- **Langage :** [TypeScript](https://www.typescriptlang.org/) (Typage strict pour données financières)
- **Style :** Tailwind CSS + DaisyUI
- **Data-Viz :** Recharts
- **IA :** Claude API integration
